# PraxisWorld Project Page

Static GitHub Pages site for:

> PraxisWorld: Object-Centric World-Action Model for Composable Robot Manipulation

The page follows the local UniT project-page structure and uses assets compiled from the latest Overleaf snapshot in `../overleaf_aot_latest`.

## Local Preview

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## GitHub Pages

Publish this directory as the root of `SC2I-HKUSTGZ/PraxisWorld` and enable Pages from the `main` branch root. The expected URL is:

```text
https://sc2i-hkustgz.github.io/PraxisWorld/
```
